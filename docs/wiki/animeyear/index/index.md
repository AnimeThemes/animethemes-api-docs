# Year Index Endpoint

The year index endpoint returns a list of unique years from all anime resources.

## URL

```sh
GET /animeyear/
```

## Authentication

None

## Parameters

None

## Response

```json
[
    1963,
    1964,
    1965,
    ...
]
```

## Example

```bash
curl https://api.animethemes.moe/animeyear/
```
