# User

---

A user API resource represents an AnimeThemes account.

## Fields

|    Name    |  Type   | Nullable | Default |  Description                 |
| :--------: | :-----: | :------: | :-----: | :--------------------------- |
| name       | String  | No       | Yes     | The username of the resource |

## Allowed Include Paths

* playlists
