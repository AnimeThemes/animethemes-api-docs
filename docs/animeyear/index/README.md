# Year Index Endpoint

The year index endpoint returns a list of unique years from all anime resources.

## URL

```sh
GET /api/animeyear/
```

## Parameters

None

## Response

```json
[
    1963,
    1964,
    1965,
    ...
]
```

<Example>

<CURL>
```bash
curl https://staging.animethemes.moe/api/animeyear/
```
</CURL>

</Example>
